📡 REST API

Эта страница считает всё в браузере. Событийный движок доступен по REST API (тот же самый набор правил):

GET  /api/schema      параметры, диапазоны, протоколы
POST /api/simulate    расчёт трассы (мгновенно, в виртуальном времени)
GET  /api/health

Также API опубликован напрямую на порту 5000. Интерактивный плеер трасс — на /latest.
